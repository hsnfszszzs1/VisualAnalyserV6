# VisualAnalyserV6

**VisionAnalyzerV6** – Advanced vision-based body analysis engine for precise 3D reconstruction, anatomical landmark detection, and metric measurements.

## Overview

This repository contains the core implementation of **VisionAnalyzerV6** (v2.9), a powerful computer vision tool designed for detailed body analysis from images. It extracts real measurements in centimeters, detects dynamic anatomical landmarks, calculates proportions, and provides rich reports.

Key features:

- Real image-driven analysis (no hardcoded values)
- Dynamic anatomical landmark detection
- Precise metric measurements (waist, hips, thighs, etc.)
- Glute projection & shape analysis
- 3D reconstruction with skeleton tracking
- Volumetric estimation
- Multi-language skill integration (English + Romanian)

## Main Files

- `vision_analyzer_v6.py` – Full implementation of the `VisionAnalyzerV6` class
- `SKILL.md` – Skill definition and activation rules for Grok

## Usage (as Grok Skill)

Activate with exact command: **Vision ON** (with an image attached)

See `SKILL.md` for full activation rules, commands, and integration details.
